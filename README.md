# 🎬 Virelume Forge

Virelume Forge is a next-generation, 1-click **Faceless Video Agent** designed for the Higgsfield ecosystem. It automates the entire process of turning a single topic/idea into 3 cinematic, vertical video clips (9:16) in parallel.

---

## ✨ Features
*   **Automatic Prompt Engineering**: Translates your core topic into unique scene prompts.
*   **Parallel Generation Loops**: Triggers image seed generation (`nano_banana_2`) and feeds them directly into video render jobs (`seedance_2_0`) simultaneously.
*   **Curated Visual Styles**: Out-of-the-box support for Cinematic Dark, Y2K Retro, Cyberpunk, and Anime aesthetics.

---

## 🛠️ Tech Stack
*   **Framework**: React 19 & TanStack Start
*   **Routing**: TanStack Router (File-based)
*   **Validation**: Zod
*   **Styling**: Tailwind CSS
*   **Core Engine**: Higgsfield MCP (AI Image & Video Generation APIs)
